# Autoshop_project

Проект по дисциплине: **"Технологии сетевого программирования"**

## Стек технологий

В разработке использовался стек **PERN**:
- **PostgreSQL** — база данных
- **Express.js** — серверная часть
- **React** — клиентская часть
- **Node.js** — среда выполнения JavaScript
